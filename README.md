## Node Backend  manage with Mongoos Database

### This is backend of task manage app develope using Nodejs, Expressjs and mongoDB

#### Technologies
- JS
- Node JS
- Express JS
- Mongo DB

#### Features
- Authentication
- Crud

#### Setup
- Clone the project.
- Create Mongo Db Account with database.
- Create ".env" file.
- Run "npm install".
- Run "npm start".
